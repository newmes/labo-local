## Why LABO

Research already happens across too many tools — a recorder for lab meetings, a reference manager for papers, Overleaf for writing, GitHub for code, and a chat app to glue it together. LABO folds that loop into one workspace, with each researcher's own AI agent connected to the room.

| | |
|---|---|
| **Meeting minutes** | Lab meetings become structured records. Action items land on the task board by themselves. |
| **Literature library** | Papers cited in conversation are captured into the room's library automatically — with PDF, discussion threads, and annotations. |
| **LaTeX writing** | Draft, revise, and compile in the same room as your references. |
| **Dev sessions** | Every researcher connects their own coding agent — Claude Code, Codex, or Cursor — and the room sees the work. |
| **Bot fleet** | Watch and steer all agents from one screen, or from your phone. |

## Local-first

Your research data lives on **your** machine. LABO installs as a desktop app and runs its backend locally — no central server holds your conversations, documents, or papers. Sharing a room with teammates is opt-in, relayed end-to-end.

**No extra AI fees.** LABO doesn't resell tokens — connect the Claude Code / Codex / Cursor subscription you already have.

## Install

Grab the installer for your OS from [Releases](https://github.com/newmes/labo-local/releases):

| OS | File |
|---|---|
| Windows | `labo-desktop-*-setup.exe` |
| macOS (Apple Silicon) | `Labo-*-arm64.dmg` |
| Linux | `Labo-*.AppImage` |

> Builds are not yet code-signed — Windows SmartScreen: **More info → Run anyway**; macOS Gatekeeper: **right-click → Open**.
