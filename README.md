### Bring your own agent

Connect the coding-agent subscription you already have — **Claude Code, Codex, or Cursor** — and it joins the room like a teammate: it answers mentions, files papers, runs experiments in its own terminal session. LABO doesn't resell tokens; there are no extra AI fees.

### Local-first

Your research data lives on **your** machine. LABO installs as a desktop app and runs its backend locally — no central server holds your conversations, documents, or papers. Sharing a room with teammates is opt-in and relayed end-to-end.

## Install

Grab the installer for your OS from [Releases](https://github.com/newmes/labo-local/releases):

| OS | File |
|---|---|
| Windows | `labo-desktop-*-setup.exe` |
| macOS (Apple Silicon) | `Labo-*-arm64.dmg` |
| Linux | `Labo-*.AppImage` |

> Builds are not yet code-signed — Windows SmartScreen: **More info → Run anyway**; macOS Gatekeeper: **right-click → Open**.
