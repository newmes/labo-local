## LABO는

- **회의록** — 랩미팅을 구조화된 기록으로. 할 일은 곧장 작업판으로.
- **문헌 서재** — 대화에서 인용한 논문이 자동으로 서재에 등록됩니다.
- **LaTeX 집필** — 문헌과 같은 방에서 초안부터 투고까지.
- **개발 세션** — 연구원 각자의 AI 에이전트(Claude Code · Codex · Cursor)를 방에 연결.
- **로컬 우선** — 연구 데이터는 내 컴퓨터에 저장됩니다. 별도 AI 요금 없이 본인 구독을 연결해 사용합니다.

## 설치

[Releases](https://github.com/newmes/labo-local/releases)에서 운영체제에 맞는 설치본을 받으세요.

| OS | 파일 |
|---|---|
| Windows | `Labo-Setup-*.exe` |
| macOS (Apple Silicon) | `Labo-*.dmg` |
| Linux | `Labo-*.AppImage` |

> 코드서명 인증서 적용 전이라 Windows SmartScreen / macOS Gatekeeper 경고가 뜰 수 있습니다 — "추가 정보 → 실행"(Windows), "우클릭 → 열기"(macOS)로 진행하세요.
