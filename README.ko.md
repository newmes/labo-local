## LABO는

연구는 이미 너무 많은 도구에 흩어져 있습니다 — 랩미팅 녹음기, 서지 관리 도구, 집필용 Overleaf, 코드용 GitHub, 그리고 이 사이를 잇는 메신저. LABO는 그 루프를 하나의 워크스페이스로 접고, 연구원 각자의 AI 에이전트를 방에 연결합니다.

| | |
|---|---|
| **회의록** | 랩미팅이 구조화된 기록으로. 할 일은 곧장 작업판으로. |
| **문헌 서재** | 대화에서 인용한 논문이 자동으로 방 서재에 등록됩니다 — PDF·디스커션·주석까지. |
| **LaTeX 집필** | 문헌과 같은 방에서 초안부터 컴파일까지. |
| **개발 세션** | 연구원 각자의 코딩 에이전트(Claude Code · Codex · Cursor)를 방에 연결. |
| **봇 함대** | 모든 에이전트를 한 화면에서, 또는 폰에서 모니터링. |

## 로컬 우선

연구 데이터는 **내 컴퓨터**에 저장됩니다. LABO는 데스크탑 앱으로 설치되어 백엔드를 로컬에서 실행합니다 — 중앙 서버가 대화·문서·논문을 보관하지 않습니다. 팀원과의 방 공유는 선택이며, 종단 간 중계로 이뤄집니다.

**별도 AI 요금이 없습니다.** LABO는 토큰을 재판매하지 않습니다 — 이미 쓰고 있는 Claude Code / Codex / Cursor 구독을 연결하세요.

## 설치

[Releases](https://github.com/newmes/labo-local/releases)에서 운영체제에 맞는 설치본을 받으세요.

| OS | 파일 |
|---|---|
| Windows | `labo-desktop-*-setup.exe` |
| macOS (Apple Silicon) | `Labo-*-arm64.dmg` |
| Linux | `Labo-*.AppImage` |

> 코드서명 적용 전이라 경고가 뜰 수 있습니다 — Windows SmartScreen: **추가 정보 → 실행**, macOS Gatekeeper: **우클릭 → 열기**.
